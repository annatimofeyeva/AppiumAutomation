# Wikipedia automation scripts

## Prerequisites

You will need the following things properly installed on your computer.

* Java
* Android SDK
* JUnit
* Inellij Idea
* Appium

## Installation
* git clone 'repository-url'` this repository
* Open the project in Intellij Idea
